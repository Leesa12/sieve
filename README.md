# Sieve Calibration System

An automated machine vision web application that replicates the sieve calibration methodology described in the paper *"An Expert System for Improving Sieve Calibration Process"* (Belan, Librantz, de Araújo — IJCA Vol 79, No.8, 2013).

This system replaces manual sieve calibration processes (which involve technicians manually measuring 300+ holes per sieve using a profile projector, taking ~3 hours) with a computerized machine vision pipeline that runs in seconds, analyzing thousands of points for superior statistical accuracy.

## Core Pipeline Architecture

The application follows the five-step workflow from the paper:
`Image Acquisition → Pre-processing → Measurement → Save the Data → Calibration Data`

1. **Scale Calibration:** Users upload a ruler or graduated scale image and click two points on an interactive HTML5 canvas. They input the real-world distance between the points to calculate the `pixels_per_mm` scale factor, which is stored in SQLite for reuse.
2. **Pre-processing:**
   - Converts the mesh image to grayscale.
   - Applies **Otsu's Thresholding** to convert to binary.
   - **Auto Polarity Check:** Automatically identifies whether holes are bright or dark. Sets the binarization so that holes are white (foreground) and wires are black (background).
   - **Dust/Noise Filtering:** Removes tiny connected components (smaller than a dynamic area threshold based on nominal size) caused by dust particles or dirt on the mesh grid.
3. **Measurement:**
   - **Hole Open Area:** Runs 8-connected component labeling to identify individual open mesh spaces. Calculates their area in mm², filters outliers (debris or tears) using the Interquartile Range (IQR) method, and computes average area, min, max, std deviation, and relative frequency histogram data.
   - **Wire Grid Thickness:** Sweeps rotation angles from -45° to +45° in 0.5° increments. For each angle, calculates the horizontal/vertical projection histograms (pixel sum profiles). Finds the optimal angle by maximizing the sum of variances of the projection profiles. Rotates the full-resolution wire image, detects grid peaks, measures peak widths at half-height (`rel_height=0.5`), and converts the pixel average to mm.
4. **Database & Persistence:** Saves calibration results to SQLite (via SQLAlchemy) linking sieves, calibration factors, and runs.
5. **Reporting:** Displays stats cards, dynamic histograms, alignment peaks, binarization step previews, and generates printable PDF certificates (using ReportLab) or CSV exports.

## How to Run Locally

### 1. Install Dependencies
Make sure you have Python 3.11+ installed. Run the following command to install the required packages:

```bash
pip install -r requirements.txt
```

### 2. Start the Server
Run the FastAPI development server with Uvicorn:

```bash
uvicorn main:app --reload
```

### 3. Open the Dashboard
Open your web browser and navigate to:
[http://127.0.0.1:8000](http://127.0.0.1:8000)
